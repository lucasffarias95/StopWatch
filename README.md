# Stopwatch - Balta.io

Projeto de cronômetro desenvolvido em C# para praticar lógica de programação e manipulação de strings.

## 🚀 Funcionalidades

* Contagem em segundos e minutos.
* Menu de opções dinâmico.
* Sistema de contagem progressiva.

## 🛠️ Tecnologias

* .NET SDK
* C# (Console Application)
* Git e GitHub

## 📝 Como rodar

No terminal, dentro da pasta do projeto, digite:
`dotnet run`
